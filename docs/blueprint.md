# **App Name**: SoundPilot

## Core Features:

- Force Media Output Redirector: Allows users to force media audio from selected apps to output exclusively through the device's internal speaker, overriding default settings.
- Floating Volume & Mute Controls: A persistent, movable on-screen widget providing quick access to a mute toggle, a volume slider, and a precise volume percentage display.
- App Exclusion Whitelist: Users can specify a list of applications that will be exempt from the force mute and redirection functionality, allowing their audio to behave normally.
- Contextual Audio Automation: Automatically disables force mute and sets the media volume to a user-defined low percentage (e.g., 7%) when wired headphones or Bluetooth audio devices are connected, reverting to previous settings upon disconnection.

## Style Guidelines:

- Primary color: #336699 (a deep, confident blue). This hue evokes a sense of control and stability, befitting a system utility app.
- Background color: #2A2F33 (a subtle dark blue-grey). This dark, desaturated tone provides an unobtrusive and modern backdrop, suitable for a utility that runs in the background.
- Accent color: #5CDBEE (a vibrant cyan). This bright, contrasting color is used for interactive elements, highlights, and notifications, ensuring visibility against the dark scheme.
- Main font: 'Inter' (sans-serif). Selected for its modern, objective, and highly legible characteristics, ideal for clear presentation of controls and data in a utility application.
- Utilize minimalist, crisp line-art style icons that are immediately recognizable for actions like mute, volume up/down, settings, and app selection, maintaining a consistent visual language.
- The floating control widget will be compact, resizable, and easily repositionable, designed for efficient single-touch interaction. Settings screens will feature clear, sectioned layouts for managing app whitelists and automation rules.
- Implement subtle and purposeful animations for UI feedback, such as soft transitions when toggling mute, adjusting volume, or opening/closing floating elements, to enhance user experience without distraction.
